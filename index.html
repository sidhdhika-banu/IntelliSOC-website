AI-Enhanced Long-Term IP Behaviour Tracking

IP-Based Long-Term User Behaviour Tracking and Risk Analysis System

Intelligent Behaviour Monitoring for Advanced Threat Detection

This showcase demonstrates how a lightweight SOC dashboard can continuously analyze user/IP activity over long time windows and transform subtle behavior changes into clear, actionable cyber risk insights.

View Research Watch Demo

About the Project

Hey this is About section sample content.

This final-year research project focuses on detecting suspicious IP and user behaviour that evolves slowly over time. Instead of relying only on instant alerts, the platform keeps behavioural memory and assigns changing risk levels, helping analysts catch stealthy attacks earlier and with better context.

Research

Hey this is Research section sample content.

Our research compares short-window security rules with long-term anomaly modelling. Initial findings show that combining temporal features, behaviour diversity patterns, and explainable scoring can significantly improve triage decisions for SOC teams handling progressive threats.

Problem Statement

Hey this is Problem section sample content.

Security teams struggle to detect suspicious IP behaviour hidden across long timelines. Fraud attempts and low-and-slow attacks often evade short-term rules, causing delayed response and high operational risk.

🕵️

Hidden Behaviour Drift

Attackers shift tactics slowly to bypass traditional threshold-based alerts.

⚠️

Fraud Detection Gaps

Isolated event monitoring misses correlated malicious patterns over weeks.

📉

Limited Long-Term Memory

Many systems lack persistent behaviour profiles and adaptive risk scoring.

Methodology

Hey this is Methodology section sample content.

The system ingests Windows and firewall logs, extracts temporal behaviour features, computes anomaly scores with lightweight ML models, and presents risk-driven alerts in a dashboard. This gives both real-time awareness and long-term behavioural context in one workflow.

Proposed Solution

Hey this is Solution section sample content.

Long-term behaviour tracking for each IP and user entity.
Dynamic risk scoring model based on temporal anomalies.
Admin dashboard for investigation and action.
Real-time alert system with explainable anomaly factors.

System Architecture

User Activity→ Behaviour Analysis Engine→ Risk Score Generator→ Admin Dashboard→ Alert System
Frontend: HTML + Tailwind Backend: Python/Node API Database: TimescaleDB/MongoDB ML: Isolation Forest, One-Class SVM
